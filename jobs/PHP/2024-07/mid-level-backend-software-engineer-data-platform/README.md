**THE OPPORTUNITY**

We are hiring a Mid-level Backend Software Engineer to our Data Platform team in Brazil!

The Data Platform Team is responsible for building and maintaining system components that empower data engineers and analysts to achieve their business objectives. Leveraging AWS cloud infrastructure alongside open-source and in-house software, our team ensures that the data solutions are self-service, cost-effective, and secure.  
Our team members come from diverse technical backgrounds, including software engineering, systems administration (focusing on Kubernetes), cloud infrastructure and services, data analysis, and data governance. Despite their varied expertise, all members are committed to cross-training and mastering all relevant activities.  
We are a well-established team with a strong culture rooted in collaboration, humbleness, accountability, empathy, and a sense of humor, fostering an environment that values teamwork and individual growth.  

Example projects:

  * Tooling around the query engine (Trino) that reduces costs 5x while improving query performance;
  * Metastore catalog manager tool to streamline table and schema evolution (GitOps + DataOps);
  * Access control system that enables ABAC access control within Trino;
  * Tooling that finds the most expensive queries and provides recommendations;
  * Solution that aggregates access logs and organizes them so queries are several times faster and cheaper to run
