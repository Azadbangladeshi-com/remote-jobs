### **About the Role:**

We are looking for a meticulous SDET who will be responsible for the design and implementation of automated tests and test environments for the Motive products. The Engineer must have a passion for diving into data, technology, be a problem solver, and have a desire to improve software quality within a team environment. This role is critical to us in ensuring technology functions efficiently and effectively, minimizes errors, and helps to meet software quality goals.

### **What You'll Do:**

  * Analyze and understand requirements, business use cases, technical requirements
  * Set up and maintain test automation frameworks.
  * Develop and implement automated tests.
  * Track and report upon testing activities, including testing results, test case coverage, required resources, defects discovered and their status, performance baselines, etc.
  * Assist in performing any applicable maintenance to tools used in Testing and resolve issues if any.
  * Update reporting tools with testing results.
  * Assist with Continuous Integration process
