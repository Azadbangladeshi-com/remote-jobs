 _ **General Summary:**_

Main Responsibilities:

1\. Will be a member of a global software development team with responsibility for developing, integrating, debugging and commercializing world class software that implements data networking protocol stacks for wireless domains - cellular (5g/4g/3g/2g encompassing 5G NR/LTE/HSPA/UMTS/EVDO/CDMA/GPRS/etc) and Wifi (802.11 flavors) as well as Bluetooth domains. Responsible for developing and maintaining wireless protocols and air interfaces to implement the relevant communication protocol specifications.  
2\. Responsible for the analyzing and troubleshooting the communication technology problems on Qualcomm platform.  
3\. Participate in wireless chip tape out, responsible for software features and protocol implementation, performance optimization, troubleshooting, etc.  
4\. Analyzes user needs and software requirements to determine feasibility of design within time and cost constraints. Designs, develops, and modifies software systems, using scientific analysis and mathematical models to predict and measure outcome and consequences of design.

Skills / Experience Required:

1\. Be able to keep abreast of the latest developments in the field of mobile communications. Familiar with wireless communication and the latest wireless technology.  
2\. Has the basic knowledge from one of below programming language: C, C++, Python, C# and other programming languages. Know the basic principles of operating system.  
3\. Understand 5G/4G/3G/2G/WLAN/Bluetooth communication standards and protocols.  
4\. 3GPP communication protocol background (NR/LTE/HSPA/UMTS/EVDO/CDMA/GPRS/etc) is preferred.  
5\. Good English communication skills. Must be able to write technical documents in both English and Chinese. must be able to read wireless application standards and technical specifications, able to work with global teams.

Education Requirements (Degree Major, GPA, Courses):  
Bachelor's degree or above in Communications, Computer Science, Electronics, etc.
