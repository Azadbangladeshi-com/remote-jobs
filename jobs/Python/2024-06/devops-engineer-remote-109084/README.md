###  **Job Description**

We are looking for an accomplished DevOps Engineer to join our growing team. You will play an important part in bridging the gap between development and operations, ensuring the smooth delivery and high availability of our applications. You will also contribute to our Site Reliability Engineering (SRE) efforts, focusing on proactive measures to improve system reliability. You will report to the Architect. This is a remote role.

 **Responsibilities:**

  *  **DevOps:**
    * Develop automated build, test, and deployment pipelines (CI/CD)
    * Manage and configure infrastructure using Infrastructure as Code (IaC) tools like Terraform or Ansible
    * Collaborate with developers to refine the software delivery process
    * Monitor and troubleshoot infrastructure and application issues
    * Identify opportunities for automation and continuous improvement
  *  **SRE:**
    * Participate in incident response and resolution processes
    * Analyze system logs and metrics to identify potential problems
    * Develop and implement tools and scripts to automate monitoring and alerting
    * Design and implement solutions to improve system reliability and scalability
    * Stay up-to-date on the latest SRE practices and methodologies

###  **Qualifications**

  * 3+ years of experience in a DevOps or related role
  * Experience with Linux systems administration
  * Experience with scripting languages (Bash, Python)
  * Proficient in CI/CD tools (Jenkins, GitLab CI/CD)
  * Experience with IaC tools (Terraform, Ansible)
  * Experience with containerization technologies (Docker, Kubernetes)
  * Familiarity with cloud platforms (AWS, Azure, GCP)
  * Understanding of monitoring and alerting tools (Prometheus, Grafana)
