 **Position Overview**

The Cloud Security Engineer role will focus on Azure and AI enablement across products as an important member of the security team and will design, implement, and maintain robust security measures within the Azure cloud environment. You will report to manager and its a hybrid work mode.

 **Responsibilities**

  * Design, implement, and maintain security controls and measures for our cloud infrastructure, with a focus on AWS and Azure platforms
  * Secure data models and databases to protect sensitive information and ensure compliance with data privacy regulations
  * Implement and enforce configuration management best practices to ensure and secure cloud environments
  * Conduct regular security assessments to identify vulnerabilities and recommend remediation strategies
  * You will develop and implement security policies, procedures, and guidelines to ensure compliance with industry standards and regulation
  * You will monitor security logs and events to respond to potential security incidents
  * Collaborate with cross-functional teams to integrate security best practices into the development and deployment processes
  * Stay up-to-date with the latest cloud security trends, technologies, and threats, and provide recommendations for continuous improvement
  * Provide guidance and support to other teams on cloud security best practices and procedures
  * Support incident response as Cloud Security SME

 **Minimum Qualifications**

  * Bachelor's degree in Computer Science, Information Security, or a related field.
  * 5 to 8 years experience as a Cloud Security Engineer, with a focus on AWS and Azure platforms.
  * Ability to program/script for automations in at least one language; Python, GO, JavaScript, Bash, or PowerShell.
  * Knowledge of cloud security best practices and industry standards.
  * Experience with GenAI technologies, LLMs and their application in cloud security.
  * Familiarity with security frameworks such as NIST, ISO 27001, and CIS.
  * Hands-on experience securing data models and databases.
  * Experience with configuration management tools and technologies, such as AWS Config, Azure Policy, or Terraform.
  * Knowledge of configuration management best practices and the ability to enforce them.
  * Experience with data encryption, access controls, and data masking techniques.
  * Hands-on experience with security tools and technologies, such as IDS/IPS, SIEM, WAF, and vulnerability scanners.

 **Preferred Qualifications**

  * Knowledge of data privacy regulations such as GDPR and CCPA.
  * Relevant certifications such as AWS Certified Security - Specialty, Azure Security Engineer Associate, or Certified Cloud Security Professional (CCSP).
  * GCP experience.
