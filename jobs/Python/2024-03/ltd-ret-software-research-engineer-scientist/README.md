# **Job Details:**

## Job Description:

Intel is hiring a motivated **RET Software Research Engineer/Scientist.** This position is in the Logic Technology Development (LTD) team, working with one of the most advanced semiconductor development facilities in the world.  
  
The LTD Resolution Enhancement Technology (RET) team is extending the limits of feature sizes that are patternable on Intel's devices, using current and future lithography manufacturing tooling. The team works closely with fab integrated lithography process and mask manufacturing teams using computational lithography tooling to optimize mask patterns and maximize manufacturing process capability. The RET team also works closely with the design rule definition teams, using lithography and patterning simulation capabilities to determine design rules for future Intel process nodes.

 **Responsibilities include:**

  * Optical Proximity Correction (OPC) simulations.
  * Image processing and statistical analysis of data for OPC models.
  * OPC model calibration
  * OPC recipe code optimization and validation.
  * Algorithmic design pattern layout manipulation.
  * Support of production mask tapeout and RET validation.
  * Creation of technical documentation and presentations.
  * Clearly and effectively communicate (both verbally and in writing) with a variety of engineers working on multiple projects.
  * Utilize technical knowledge to independently identify, analyze and solve technical problems.
