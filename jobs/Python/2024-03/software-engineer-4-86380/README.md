About The Position

The **Software Engineer in Test** plays a crucial role in validating and automating test cases for Networking Products. This highly technical position requires proficiency in networking fundamentals, Quality of Service (QoS), firewall technologies, L2/L3 protocols, virtualization, and test frameworks.

Key Responsibilities

  * Qualify platform features.
  * Design high-quality test plans based on requirements.
  * Set up the necessary network topology for testing.
  * Demonstrate strong analytical skills.
  * Execute test cases and develop automation scripts.
  * Provide weekly progress reports.

Desired Technology Knowledge

The ideal candidate should have an advanced level of knowledge and practical hands-on experience in the following technologies:

  * **PFE Packet Forwarding**
  * **Dynamic Load Balancing**
  * **Load Balancing**
  * **ECMP (Equal-Cost Multipath)**
  * **Policer**
  * **Firewall**
  * **QoS/COS (Quality of Service/Class of Service)**
  * **Link Aggregation**
  * **L2 Switching**
  * **VLANs**
  * **Q-in-Q**
  * **L2 Protocols** (e.g., xSTP, LACP, LLDP, VVRP)
  * **Switching Architecture**
  * **EVPN-VXLAN**
  * **MC-LAG (Multi-Chassis Link Aggregation)**
