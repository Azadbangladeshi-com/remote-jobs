# **Job Details:**

## Job Description:

Architects, designs and provides technical oversight and direction of the development of cloud software for all innovation projects in the systems architecture group in the office of the CTO including system software, frameworks, protocols, algorithms, and applications, as well as software structure and validation direction. Researches and designs optimal end to end cloud computing software solutions for cloud, AI, orchestration, enterprise, microservices, and/or distributed computing. Engages with major customers to understand use cases, participates in open source to influence architectures and optimize software, and interacts with technologists, developers and product/product managers across Intel to evaluate feasibility of requirements and determine priorities for development. Makes design choices focused on hardware adaptability, manageability, scalability, usability, and resiliency for the software structure, protocols, and algorithms.
