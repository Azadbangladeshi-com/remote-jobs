# **Job Details:**

## Job Description:

At Intel our purpose is to create world-changing technology that improves the life of every person on the planet.

  
Within the Software and Advanced Technology Group (SATG) we enable Intel's vision for software and advanced technology by delivering open and secure software platforms. We are on a mission to help enterprise customers secure their applications, data and critical infrastructure through best-in-class software and hardware technologies. Our decentralized trust platform, a scalable high performance geo-distributed ledger built on BFT and consensus algorithms, operates as a cloud-based SaaS and on-premises solution with deep security technology enabling next generation of data and compute protection for our customers.

  
We are looking for a Staff Software Validation Engineer to join our Trusted and Security Products Engineering team. This is a career defining role which will contribute directly to Intel's transformation through strategic software contributions. You will be part of our core validation team under quality engineering developing validation strategies, testing, and contributing towards delivering high-quality secure software for our customers. If you enjoy being part of a world class software engineering team focused on delivering next generation highly secure software solutions, this role is for you.  
  
Responsibilities:

  * Designs software validation strategies, develops tools and methodologies to validate Intel software product features spanning various deployment environments (on-prem, cloud, hybrid) and solutions (SaaS).

  * Designs and develops automation tests using Python framework.

  * Contributes towards enhancing automation frameworks and tools.

  * Works closely with cross functional leads and ensures functional and system testing plans are developed following risk-based approach.

  * Focuses on 100 percent automation strategy, where all tests are automated with stability prior to running them.

  * Leverages latest technologies like ML and Gen AI in day-to-day work increasing developer productivity.

  * Possesses continuous innovation and improvements mindset driving team's technical growth.
