  * Work across multiple teams and products to define and implement strategy, driving consistency, usability, and governance of our data assets. 

  * Ensure detailed understanding of metadata, usage, standards, and quality. 

  * Automate data lineage extraction process from any code base (Java/Python/Spark/ETL tools/Unix Scripts)
