### **About the job:**

Hims and Hers is seeking a Software Engineer to help build a fast, reliable, and frictionless telemedicine experience. This software allows Clinicians to communicate with patients, see their medical history and any relevant information to make an informed decision for treatment. Our stack consists of Kotlin and Postgres along with a (growing) list of AWS services.
