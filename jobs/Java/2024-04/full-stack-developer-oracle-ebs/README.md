### Job Description

The IT Engineer III role is to lead and participate in development activities and on-going support related to Oracle E-Business Suite. This individual will apply problem solving skills, knowledge of best practices and agile methodologies to gather requirements, solution design, development, and testing. The individual will be responsible for following job functions

 **Responsibilities**

  * Participate and lead the analysis, development, testing and deploying of software applications using the current set of standards and methodologies.
  * Convert stories and specifications into high quality working code.
  * Prepare unit test plan and unit test cases.
  * Perform unit testing of programs according to defined process using the unit test plan and unit test cases.
  * Perform quality and function reviews during the development phase.
  * Develop detailed estimates for design and development phases based on stories and requirements.
  * Facilitate the clarification of story issues as they arise during the development process.
  * Coordinate with the technical team, Oracle support, DBA, IT infrastructure for implementing enhancements/SR resolution.
  * Provide on-going support to Oracle E-Business Suite and related custom and 3rd party software packages on a rotational basis.
  * Facilitate and/or participate in code reviews and application tuning activities.

### Qualifications

  * 5+ years of development experience in Oracle tools such as SQL, PL/SQL, Reports, Forms, Workflow
  * Strong experience with Oracle functions, procedures, triggers, packages & performance tuning
  * Experience with Oracle database version 12c, 19c
  * 5+ years of Knowledge/experience in Oracle e-Business Suite (EBS) R12 including modules such as Order Management, Install Base, Pricing, Configurator, Inventory, Quoting, WIP, Payables, Purchasing and Receivables.
  * Good understanding of relational database design and normalization.
  * Experience in Application and SQL tuning.
  * Ability to produce clean, readable and efficient code that is easy to maintain.
  * 2+ years of development experience in Java. Experience in other object-oriented programming language(s) is a plus.
  * 2+ years of development experience in Oracle Applications Framework (OAF).
  * 2+ years of development experience in XML Publisher.
  * 2+ years of development experience in Oracle ADF technology.
  * Experience in Unser Interface design.
  * Experience in Web Services using SOAP, XML, JSON, Restful Services preferred.
  * Knowledge of the software development lifecycle and agile methodologies (Scrum).
  * Knowledge on object-oriented development.
  * Experience with MuleSoft or comparable iPaaS product a plus.
  * Experience in UI development using Angular a plus.
