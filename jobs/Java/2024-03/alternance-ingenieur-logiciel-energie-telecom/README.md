### Description du poste

### Votre futur environnement de travail

Au sein d’un projet en cours de développement pour un grand compte français du secteur de l’énergie, la mission consiste à concevoir et intégrer un agent conversationnel capable de répondre en langage naturel aux questions des utilisateurs, afin d’effectuer un premier niveau de support.

### Contexte technique

Applications web avec un frontend Angular et un backend Java, déployées en hybride sur les datacenters clients et sur le Cloud public AWS.

### Mission

En tenant compte des contraintes du projet, concevoir une solution permettant de réaliser le support de niveau 1 de façon automatique, sans l’intervention d’un agent humain. Cet agent conversationnel devra être capable de répondre en langage naturel aux différentes interrogations simples des utilisateurs du site web. Il devra être intégré dans le code de l’application existante, et déployé dans l’environnement client.
