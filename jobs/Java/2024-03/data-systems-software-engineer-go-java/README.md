## About the Team / Role:

The Virtual Services Gateway (VSG) team is responsible for designing, developing and managing an infrastructure of applications and services used to collect data about cable modem performance and usage. Our platform is used to help monitor network reliability and inform complex decisions to improve our customer's internet experience. We're a global team that's passionate about solving problems through collaboration and innovation.

As an engineer on the team, you'll be at the forefront of developing highly scalable, critical software solutions. Our team owns a wide variety of systems, ranging from REST API microservices to streaming data pipelines capable of processing hundreds of thousands of messages per second. We work with exciting technologies throughout the stack including a myriad of languages like Go, Java and Python, numerous data processing technologies like Kafka, Flink and Spark, and we extensively use AWS services like S3, Athena, and RDS.
