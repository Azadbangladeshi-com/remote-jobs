En immersion totale au sein de l’équipe technique, vous recueillerez et réaliserez les besoins utilisateurs dédié à des projets pour des acteurs majeurs de ces domaines (assurances, mutuelles, grands groupes de protection sociale, régimes spéciaux de retraite).
