### Responsibilities:

* Perform Privileged Access Management functions
* Perform User and Privilege Access Administration functions
* Implement and Support the Identity Access Management solutions
* Manage applications and coordinate systems integration
* Perform user administration on all applications, such as web-based applications and thick client applications etc.
* Grant application function access/role to users based on duly authorized requests
* Develop procedures to maintain access security and protect systems from unauthorised use, acts of nature and user abuse.
* Perform Recertification of IDs and Accesses
* Initiate user access recertification on a regular basis to ensure no obsolete account or excessive access granted
* Involve in projects which include managing vendors, testing, implementation support, etc.
* Ensure all security solutions are working and comply with company policy and procedures. 
* Respond and remediate timely on any service outages with customer satisfaction as priority.
* Operate independently and accomplish engineering and operation tasks within the agreed timeline.

* Strong analytical, organizational, and problem-solving skills are required.
* Excellent communication skills.
* Must be highly flexible and adaptable to change.
* CISA and PMP are considered as a plus.
