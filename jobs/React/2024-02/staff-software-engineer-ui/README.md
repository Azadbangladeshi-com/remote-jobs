You are an experienced engineer that is passionate about building great user experiences, is generally excited about new front-end technologies, and is ready to push the boundaries of what is achievable. ### Qualifications

 **To be successful in this role you have:**

  * 9-13 years of software industry experience mainly in Front-end roles
  * Passion for software development and problem solving
  * Significant experience designing and implementing highly interactive user interfaces for high volume web applications
  * Expertise working with JavaScript libraries and concepts such as Angular, React, NPM, Node.js, Web Components, Virtual DOM
  * Strong written and verbal communication skills
