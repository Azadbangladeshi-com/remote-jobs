# **THE OPPORTUNITY**

We are hiring a **Mid-level Frontend Software Engineer** for our CODE - Retention team!  
CODE is an acronym for Consumer Digital Experience, which means we're in charge of the experience and services that will guide people's wellbeing journey. Our main goal in CODE is to make wellbeing universal. From knowing about Gympass and getting access to exclusive benefits to using our vast partner's network, it is in the end-user experience.  
CODE teams are responsible for developing and maintaining Gympass's App, frontend, and backend services, and Design System and work closely with product, UX, and data teams to create an excellent experience for the users of our platform.  
Our mission as a Retention team is to reduce the overall churn by recommending partners and apps, helping users explore their plans, and showing them the Gympass value.
