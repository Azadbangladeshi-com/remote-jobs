### Job Description

Are you a web development guru who is able to do mobile development while thriving on building seamless cross-platform experiences? We're searching for a passionate and skilled Mid to Senior level Web and Mobile Developer to join our dynamic and collaborative team. You'll be instrumental in crafting next-generation web and mobile experiences that captivate users, while pushing creative boundaries.

### What You'll Do:

  * Architect, design and develop high-performance mobile web applications for iOS and Android, using React Native.
  * Write clean, maintainable and testable code that adheres to best practices.
  * Integrate with various APIs and backend services.
  * Collaborate closely with Designers, Product Managers and other Developers to bring ideas to life.
  * Stay up-to-date with the latest mobile technologies and trends, making recommendations for new tools and technologies to help us build better mobile applications.
  * Strong attention to detail placing the end user at the forefront of all decisions.
  * Contribute to the continuous improvement of our development processes and methodologies.

### Qualifications

  * 4+ years of experience as a Mobile Web Developer.
  * Strong command and proficiency in Javascript and Typescript. 
  * Proficiency in React-Native, ReactJS. 
  * Solid understanding of iOS and Android development, preferably with experience in native languages (Swift/ Kotlin).
  * Experience with state management and asynchronous programming.
  * Strong problem-solving and analytical skills.
  * Excellent communication and collaboration skills.
  * A passion for building high-quality, user-centric products.
  * You value and have opinions on testing and the confidence it adds to project development. 
  * Bonus points for experience with CI/CD pipelines.
