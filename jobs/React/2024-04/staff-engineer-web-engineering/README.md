### **Minimum Qualification**

  * Bachelor's degree in computer science, computer science engineering, or related field required.
  * 8+ years of active engineering and/or architecture experience in a complex environment
  * Advanced level experience in React, JavaScript Frameworks, HTML5, CSS3
  * Strong understanding of web application ecosystem and end-to-end architecture
  * Experience creating and applying strong API design principles
  * Exhibit skills to analyze, optimize, fine-tune and benchmark service performance
  * Experience in CI/CD tools like Github Actions, Jenkins.
  * Excellent verbal and written communications skills with the ability to create and deliver architecture detail & vision across the organization.
  * Experience with WAI-ARIA (Web Accessibility Initiative) and WCAG (Web Content Accessibility Guidelines).
  * Experience with reactive programming using frameworks like vert.x, spring reactive are a plus.
