### **Job Description**

Senior Platform Operations Engineer will be a key member of our DevOps Team, helping to build and support the core infrastructure of the GolfNow Platform services and products through activities and key responsibilities that include:

  * Contributing to efforts that ensure the continuous and smooth running of the GolfNow platform while serving a large volume of traffic.
  * Leveraging Amazon Web Services to build highly available services.
  * Developing Infrastructure as code using tools like Terraform.
  * Developing configuration management/automation/deployment pipelines using tools like Ansible.
  * Working closely with the NBC Sports Next Cyber Security Team to maintain and improve the security of the Platform.
  * Contributing to and using our GitHub Pull Request-centered development pipeline as we continuously deliver value to our customers.
  * Using tools such as Datadog and Splunk to monitor the health of the platform.
  * Being an advocate for quality code and engineering practices that enable Continuous Delivery.
  * Participation in a sustainable on-call schedule.
